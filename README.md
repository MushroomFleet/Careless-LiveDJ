# CARELESS-LIVEDJ

**Cross The Streams** — a dual-stream Gemini Lyria Live DJ desktop application for Windows. 

![Cross The Streams](https://raw.githubusercontent.com/MushroomFleet/Careless-LiveDJ/refs/heads/main/69EF2F5A-AF95-4026-8F34-388CA3321FE4.png)

Two simultaneous Google Gemini **Lyria RealTime** music generation streams, mixed through a real DJ-style crossfader, with a sample pad, beat-synced decks, MIDI control, automation lanes, FX, and continuous master recording. Bring your own API key.

---

## ✨ What It Does

CARELESS-LIVEDJ runs **two independent Lyria Live music generation sessions side-by-side** and lets you perform with them like a real DJ rig:

- **Dual Lyria streams (Deck A / Deck B)** — each deck runs its own prompt cache, parameter panel, and live music session
- **Crossfader mixing** — blend between the two AI-generated streams in real time
- **Beat sync engine** — beat tracker + time-stretch worklet keep the decks locked together
- **Sample pad** — trigger one-shots, record from mic, route through an echo chain
- **File decks** — drop in audio files alongside the Lyria streams
- **Track library** — import, search, and recall your own audio
- **Music gallery** — browse, tag, and recall generated sessions
- **MIDI control** — learn-mode binding for any control surface, persisted across sessions
- **Automation lanes** — record and replay parameter automation
- **Master FX panel** — output-stage effects on the master bus
- **Continuous master recording** — never miss a take
- **Debug console + spectrograph** — full visibility into the audio graph
- **Dark / light themes**

## 📥 Installation (Recommended)

The easiest way to use CARELESS-LIVEDJ is to download the prebuilt **MSI installer** from the [Releases](https://github.com/MushroomFleet/Careless-LiveDJ/releases) page.

1. Download `CARELESS-LIVEDJ_x.x.x_x64_en-US.msi` from the latest release
2. Run the installer (Windows 10 / 11, 64-bit)
3. Launch **CARELESS-LIVEDJ** from the Start menu
4. Open **⚙ SETTINGS** and paste your **Google Gemini API key**
5. Press a deck's **CONNECT** button and start generating

> **Note:** Only the MSI installer is distributed in this repository's releases. Source code is not published here — see the developer section below if you want to build it yourself.

### Getting an API Key

CARELESS-LIVEDJ is a "bring your own key" application. You'll need a Google Gemini API key with **Lyria RealTime** access — get one from [Google AI Studio](https://aistudio.google.com/).

Your key is stored locally in an encrypted vault and is never transmitted anywhere except directly to Google's API endpoints.

## 🛠 For Developers

The full source code is **not published in this repository**. Instead, we ship the architecture as a **TINS plan** — a model-agnostic, This-Is-Not-Software specification that any capable coding harness can build from.

### Building from the TINS plan

The included [`stage0-CARELESS-LIVEDJ-TINS.md`](stage0-CARELESS-LIVEDJ-TINS.md) is the **foundation specification** for the application. To build it yourself:

1. Clone or download this repository
2. Open `stage0-CARELESS-LIVEDJ-TINS.md` in your coding harness of choice (Claude Code, Cursor, etc.)
3. Use a **strong frontier coding model** — we recommend Claude Opus 4 / Sonnet 4 or equivalent. Smaller models will struggle with the architectural scope
4. Let the harness scaffold the project, then iterate stage-by-stage
5. To produce a Windows MSI build matching the released installer, run `npm run tauri:build` once the implementation is complete

### Stack at a glance

- **Frontend:** React 18 + TypeScript + Vite, Zustand stores, CSS Modules
- **Desktop shell:** Tauri 2 (Rust)
- **Audio:** Web Audio API, AudioWorklet time-stretching (SoundTouch.js)
- **AI:** `@google/genai` SDK — Lyria RealTime live music generation
- **Persistence:** Tauri SQL plugin + filesystem vault
- **MIDI:** Web MIDI API
